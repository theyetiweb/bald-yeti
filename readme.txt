== Description ==

The Bald Yeti WordPress theme is purpose-built for Headless WordPress. It is intended to be used in conjunction with a separate front-end application and does not contain any front-end assets.

Refer to the [GitHub repository](https://github.com/theyetiweb/bald-yeti/releases) for more information on version history and updates.
